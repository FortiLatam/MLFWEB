1.2 Lubuntu Setup

Lubuntu client is running as a live CD ISO image, and needs a little tweaking. Run this every time you restart Lubuntu client.

  • Connect via FortiPoC WebUI > Client > DISPLAY to the Lubuntu client running inside FortiPoC.
  • Open a LXTerminal window (Start menu > System Tools)
$ cd Downloads
$ wget http://10.2.2.254/static/docs/latest/lubuntu-setup.sh
$ sudo bash lubuntu-setup.sh

The Lubuntu client is now reachable by ssh, the password is fortinet
